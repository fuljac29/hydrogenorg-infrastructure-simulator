# HydrogenOrg Infrastructure Simulator - Layer 1

Development model for the first HydrogenOrg project.

## Run

```cmd
pip install streamlit pandas numpy
streamlit run Hydrogenorg_Infrastructure_Simulator_Layer1.py
```

Or double-click:

```cmd
OPEN_HIS_LAYER1.bat
```

This is a conceptual research simulator, not an engineering-certified model.
